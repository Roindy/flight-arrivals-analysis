# flight-arrivals-analysis
Скрипт, который достаёт из базы авиаперелётов три отчёта и сохраняет в Excel.

## Что внутри
- Топ аэропортов по количеству прилётов
- Прилёты по месяцам (для каждого аэропорта)
- Пассажиры, которые чаще всего прилетают в каждый аэропорт

## Как работает
1. Подключается к PostgreSQL через SQLAlchemy
2. Выполняет три SQL-запроса
3. Сохраняет результат в `report.xlsx` на рабочий стол

## Запуск
```bash
pip install pandas sqlalchemy psycopg2-binary
python main.py
```

## Данные
Использована учебная база данных об авиаперевозках по России от Технологической лаборатории экономического факультета НГУ.

- **Сайт:** [datalab-nsu.ru/edudb](https://datalab-nsu.ru/edudb)
- **Вес архива:** 237 МБ
- **Вес после разворачивания:** 909 М
